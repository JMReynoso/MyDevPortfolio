# My dev portfolio

This is a code bundle for My dev portfolio. The original project is available at https://www.figma.com/design/U3Yp4YGXDyB0vzqw469tvq/My-dev-portfolio.

I wanted a portfolio with a minimalistic, warm, and slightly rustic design. So, with the help of Figma Make, I was able to make my vision a reality. 

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

Run `docker compose -f docker-compose.dev.yml up --build` to start docker development server
